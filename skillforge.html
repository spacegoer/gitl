// About Us

GiTL-SkillForge

A comprehensive three-phase employability skill development program for Information Technology students, offering 11 certification tracks.

Through live and recorded sessions, advanced technology boot camps, and a 6-week internship, SkillForge equips you with advanced technical skills such as programming logic & algorithms, data structures, and web development as well as enhancing your analytical, problem-solving and business communication skills.

By the end of the program, students gain adequate technical proficiency that will help them navigate the challenges of modern recruitment.

About Learn Square Technologies

The SkillForge program is being offered by Learn Square Technologies (LST) in association with GCTU-Infinity Tech Limited (GiTL).

Learn Square Technologies comprise a team of experts who has a collective experience of more than 60 years in the field of Edutech, Software consulting and development, Healthtech, Telecom and Life Science, who have trained and mentored more than 50 thousand students in various technologies, problem solving, logical thinking and soft skills in the last 15 years.

LST’s vision is to revolutionize the Educational landscape and empower every student with their state-of-art technology platform powered by AI.

// Phase 1

Foundational skills development

This phase focuses on building core technical and communication skills. Students participate in live boot camps and recorded courses covering:

  • Programming Logic: Boot camps that help students develop problem-solving abilities, improve algorithm design, and enhance debugging skills.
  • Business Communication & Technical Writing: Recorded courses aimed at improving written and verbal communication, essential for professional success.
  • Java Programming: A comprehensive boot camp that teaches Java syntax, object-oriented programming, and application development.
  • GitHub Workshop: Hands-on experience with GitHub, where students learn version control and collaboration best practices.
//Phase 2

Advanced Technical and Analytical Skills

In this phase, students deepen their knowledge of technical subjects while honing their problem-solving and analytical skills:

  • Data Structures and Algorithms (DSA) through Java: Focus on implementing complex data structures and algorithms, optimizing performance.
  • Mathematical Aptitude & Logical Reasoning: Recorded courses that enhance analytical skills and logical reasoning.
  • React/Bootstrap for Web Development: Live boot camps focused on frontend web development and integrating APIs.
  • Emerging Technologies: An exploration of cutting-edge technologies like AI, IoT, and blockchain.
// Phase 3

Hackathon-Style Virtual Internship

The final phase is a 6-week hackathon-style internship where students apply the knowledge gained from previous phases to real-world projects. Teams work together to develop full-cycle applications using Java and React, simulating an industry-like experience. This phase emphasizes:

  • Collaborative project work in a time-bound environment
  • Mentorship from industry experts.
  • Presentations to showcase projects, refining students' technical communication and problem-solving skills.
